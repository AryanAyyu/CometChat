### 2. What CometChat UI Kit Builder Does NOT Have
Explains missing features like:

- No moderation components  
- No analytics UI  
- No backend templates  
- No custom message builder  
- No workflow automation  

### 3. Cons / Limitations
Covers drawbacks such as:
- Re-export overwrites edits  
- Limited customization  
- Platform inconsistency  
- Missing advanced workflows  

### 4. Suggestions / Improvements
Actionable enhancements like:
- Admin UI kit  
- Role-based UI generator  
- Modular export  
- Full-stack templates  
- Custom message renderer builder  
- Workflow builder
